vLCD_CLEAR ( void  )

Function to clear the LCD Display.

Function is called to clear all data displayed on the LCD. Also returns the cursor to the home location on the top row. The command used to clear the display takes 1.53ms to run. Proper time needs allocated to ensure no commands are sent to the display for that time

[in] nothing

Returns
nothing
